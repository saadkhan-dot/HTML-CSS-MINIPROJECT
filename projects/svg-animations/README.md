# 🎬 SVG Animations & Transitions Demo

A complete showcase of SVG animation techniques using CSS and SMIL.

## 🎯 Purpose
To explore:
- SVG shapes
- CSS transitions
- Keyframe animations
- SMIL animations
- Transform animations

## 🔥 Included Animations
- Hover transition
- Move animation
- Color change
- Scale (pulse effect)
- Rotation
- Path drawing
- Fade effect

## 🛠 Technologies Used
- HTML5
- CSS3
- SVG
- SMIL Animations

## 🚀 Learning Outcome
Understanding the difference between:
- CSS-based animation
- SVG animate
- animateTransform
- Keyframe animations

Built as a visual animation reference project.
