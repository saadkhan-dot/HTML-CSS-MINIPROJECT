# 💘 Happy Valentine’s Day – Canvas Animation

An animated Valentine-themed webpage built using the HTML5 Canvas API and JavaScript.

## 🎯 Purpose
This project demonstrates:
- Canvas drawing
- Particle systems
- Animations using requestAnimationFrame
- Gradient backgrounds
- Custom heart shapes

## ✨ Features
- Floating heart particles
- Firecracker burst effects
- Animated Valentine greeting text
- Gradient background styling

## 🛠 Technologies Used
- HTML5 Canvas
- JavaScript (Vanilla)
- CSS3 (Gradients, Shadows)
- Google Fonts

## 🚀 Learning Outcomes
- Understanding animation loops
- Drawing shapes using bezier curves
- Managing particle arrays
- Combining CSS styling with Canvas visuals

Built as a creative canvas experiment.
