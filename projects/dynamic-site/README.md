# 🎨 Notion-Inspired Canvas Demo

A minimal landing page with gradient background and basic canvas drawings.

## 🎯 Purpose
This project explores:
- Linear gradient backgrounds
- Button styling
- Canvas drawing basics
- Simple illustration using Canvas API

## ✏️ Canvas Elements
- Stick figure drawing
- Checkmark box
- Styled UI layout

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript (Canvas API)

## 🚀 Learning Focus
- Canvas context methods
- Stroke styles and line width
- UI + Canvas combination
