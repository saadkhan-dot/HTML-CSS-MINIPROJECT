# 📚 BookNest – Simple Book Showcase

BookNest is a basic book showcase webpage built using pure HTML.  
It displays featured books using tables and images while maintaining a clean layout.

## 🎯 Purpose
This project focuses on:
- HTML page structure
- Headings and semantic tags
- Image embedding (external + base64)
- Table-based layout practice

## 🛠 Technologies Used
- HTML5
- Inline styling (initial version)

## 📌 Features
- Featured books section
- External and embedded images
- Simple call-to-action button

## 🚀 Future Improvements
- Convert table layout to Flexbox/Grid
- Add external CSS styling
- Improve semantic structure
- Add navigation links

Created as part of foundational HTML practice.
